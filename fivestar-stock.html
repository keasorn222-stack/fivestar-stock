กรุณาตั้งค่า Google Apps Script URL
📋 ขั้นตอนการตั้งค่า:
  1. เปิด Google Sheet → Extensions → Apps Script
  2. วางโค้ดจากไฟล์ Code.gs แล้วบันทึก
  3. กด Deploy → New deployment → Web app
  4. ตั้ง "Who has access" เป็น Anyone
  5. Copy URL ที่ได้มาวางด้านล่าง
